# Robot Architecture Diagrams

This directory contains architectural diagrams for the humanoid robot system.

## Diagram Types

- **System Architecture**: Overview of the complete robot system
- **Control Flow**: Data flow between different control components
- **Hardware Layout**: Physical arrangement of sensors and actuators
- **Software Stack**: Software components and their interactions

## File Formats

- `.svg` - Vector diagrams (preferred for scalability)
- `.png` - Raster diagrams (for complex images)
- `.drawio` - Editable diagrams (for Draw.io)

## Naming Convention

```
[component]_[type]_[version].[format]
```

Example: `control_system_arch_v1.svg`
